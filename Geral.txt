Requisitos Obrigatórios:
1 - Página que mostre os cases cadastrados.
2 - Admin para Cadastro de Cases em Drupal

3 - Criar uma API com swagger.
4 - Enviar o código fonte via link do repositório do github pronto para rodar com docker, preferencialmente havendo um docker-compose para subir uma instância para aplicação e outra para o banco de dados.
